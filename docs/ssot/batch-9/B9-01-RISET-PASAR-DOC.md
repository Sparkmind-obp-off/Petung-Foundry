## 1. Glosarium kanonik (istilah niche)

| Istilah | Arti singkat | Relevansi produk |
|---|---|---|
| **Weton** | Hari lahir Jawa = gabungan hari (7) + pasaran (5) — siklus 35 hari. | Input utama personalisasi. |
| **Neptu** | Nilai numerik weton (hari + pasaran). | Mesin perhitungan deterministik. |
| **Primbon** | Kitab kearifan Jawa: watak, jodoh, rezeki, hari baik, dll. | Sumber konten edukasi-budaya. |
| **Pétungan / Pétung** | "Perhitungan" Jawa (mis. hari baik hajat). | **Nama brand** (B9-02). |
| **Pasaran** | 5 hari pasar Jawa: Legi, Pahing, Pon, Wage, Kliwon. | Komponen weton. |
| **Hari baik** | Tanggal terpilih untuk hajat (nikah, pindahan, buka usaha). | Outcome inti yang dijual. |
| **Kejawen** | Spiritualitas/filosofi Jawa. | Konteks budaya (bukan agama formal). |
| **Hari joyo / dino apik** | "Hari baik/jaya" hasil pétungan jodoh. | Outcome Pétung Pengantin. |

> **Catatan framing:** semua produk diposisikan sebagai **kearifan lokal & edukasi-budaya** +
> personalisasi, **bukan** klaim ramalan pasti (lihat B9-04 Truth-Lock & Etika).

### 2.3 Demand berulang per momen hajat (kenapa LTV bisa naik)

| Momen hajat | Frekuensi *est.* | Outcome Pétung relevan |
|---|---|---|
| Pernikahan | ±1,48 jt/thn nasional; mayoritas keluarga Jawa pakai weton | **Pétung Pengantin** (beachhead) |
| Buka usaha / cabang baru | UMKM ID ~65 jt unit; ratusan ribu pendaftaran/tahun | **Pétung Usaha** (jembatan ke BarberKas/Toko Online) |
| Pindahan rumah / boyongan | umum, musiman | **Pétung Hajat** |
| Sunatan / aqiqah / syukuran | umum, berulang per keluarga | **Pétung Hajat** |
| Ulang tahun / penasaran weton | kapan saja (impulse) | **Kartu Weton** (tripwire) |

> Satu pelanggan bisa kembali untuk **beberapa momen seumur hidup** → LTV ladder (B9-03 §4).
